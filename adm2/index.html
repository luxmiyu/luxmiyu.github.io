Any Dimension Mod 2

Create dimensions out of any block!

Create dimensions out of any block! This mod adds 400+ dimensions to the game, one for each of the possible blocks. Build a portal out of a block and light it with the Any Dimensional Portal Wand! Now compatible with Immersive Portals!

Also has a couple useful wands to use in creative mode for demonstration purposes. Download the mod from any of the links below:

Manual Installation

1. Create or select a profile. Make sure it's a compatible mod loader and Minecraft version. You can view the mod's released versions here.

2. Click on the "Install" button on the mod's page. The Modrinth App will automatically download and install the mod for you.

3. Make sure you have the required dependencies installed. Starting from Minecraft 1.21, you will need Architectury API. If you're using Fabric or Quilt, you will also need either the Fabric API or Quilted Fabric API, respectively.

4. (Optional) If you're using Fabric, you can also install Mod Menu as well as Immersive Portals (if compatible with your Minecraft version), to make the mod experience even better.

5. Launch the game on Modrinth with the profile you installed the mod on.

6. Find your Minecraft mods folder. Usually located at %appdata%/.minecraft/mods. If unsure, just type that in your file explorer's address bar and it should take you there.

7. Move the downloaded mod and dependencies into the mods folder.

8. Launch the game with the mod loader you installed.

There are two major ways to extend the features of Any Dimension Mod 2.

Immersive Portals

Any Dimension Mod 2 is compatible with Immersive Portals! You can create portals out of any block and travel between dimensions seamlessly. This makes use of Immersive Portals' data pack support, which means it does not depend on any specific version of Immersive Portals.

Expansion Data Packs and Mods

Any Dimension Mod 2 works with blocks from other mods using data pack and mod expansions! If you'd like to create your own data pack or mod expansion, check out the GitHub page for Any Dimension Mod 2 Data Pack Expansion, as well as the example mod provided, Any Dimension Mod 2 Expansion - Applied Energistics 2.
